# WokSpec

AI-native products and infrastructure for creators, operators, and small teams.

Build a site. Launch a product. Add AI. Ship tools people actually use.

[wokspec.org](https://wokspec.org) · [Community](https://github.com/WokSpec/Community)

## Live products

| Product | Live | What it is |
|---|---|---|
| **WokSpec** | [wokspec.org](https://wokspec.org) | Main site, portfolio, status, and client dashboard |
| **Eral** | [eral.wokspec.org](https://eral.wokspec.org) | AI runtime: Worker API, embeddable widget, and browser extension |
| **WokGen** | [wokgen.wokspec.org](https://wokgen.wokspec.org) | AI pixel art studio for sprites, tilesets, animations, and game assets |
| **Vecto** | [vecto.wokspec.org](https://vecto.wokspec.org) | AI design studio for vectors, brand kits, UI-UX, voice, and creative assets |
| **Dilu** | [dilu.wokspec.org](https://dilu.wokspec.org) | No-code launch platform with production-ready templates and integrations |
| **WokTool** | [tools.wokspec.org](https://tools.wokspec.org) | 80+ browser-based tools for developers and designers |
| **WokPost** | [wokpost.wokspec.org](https://wokpost.wokspec.org) | AI-curated news platform with ranked feeds and editorial analysis |
| **Chopsticks** | [chopsticks.wokspec.org](https://chopsticks.wokspec.org) | Discord bot for music, moderation, economy, and AI |
| **WokAPI** | [api.wokspec.org](https://api.wokspec.org) | Core auth, billing, sessions, and AI routing infrastructure |
